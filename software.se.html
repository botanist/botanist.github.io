Programvaran

Alltid tillgängligt. På resande fot eller hemma i soffan

Du kommer åt din installation via vanliga webbläsaren i din PC eller via appen i din smartphone. Då all data lagras och bearbetas på våra maskiner kan du komma åt denna från var som helst i världen. Utmärkt för att kolla att grannen vattnar som lovat när du är på semester.

En enkel app. Alltid i fickan

Via vår app till iPhone eller Android kan du se vad som behöver åtgärdas, få en snabb överblick av hela din miljö, detaljerad information om enskilda växter, göra enklare inställningar, godkänna nya enheter och mycket mer.

Med smartphonens kamera läser du enkelt av QR-koden och får på så sätt direkt upp detaljer om just den enheten.

Växtdatabas för skötsel. Inga mer gissningar

Om du inte vill ställa in sensorerna manuellt kan du välja från vår växtdatabas. Den innehåller optimala värden för t.ex. temperatur, ljus och fuktighet. Och om du sår frön kan vi hålla koll även på de olika fasernas optimala värden.

Genom att betygsätta dina växter kan vi löpande förbättra våra rekommendationer utifrån riktiga mätningar.

Historisk data. Analys på ett enkelt sätt

Utifrån rådatan som sensorerna samlar in räknar vi inte bara ut aktuella värden utan också medelvärden över timmar, dagar och veckor. Du kan enkelt filtrera ut perioder, se enskilda värden, jämföra data mellan sensorer eller se hur det skulle passas med någon annan växt ur vår växtdatabas.

Du kan få det presenterat både som snygga diagram eller tabeller. Vill du göra mer avancerad analys själv går det självklart att exportera till CSV som enkelt kan importeras i MS Excel eller liknande.
